# SAVRA Dashboard

This is a full stack dashboard project made for analyzing teacher performance.

The dashboard shows teacher insights, weekly performance charts and detailed data in table format.  
User can select a teacher and view their complete analysis.

---

## Tech Stack

Frontend:
- React (Vite)
- CSS
- Recharts

Backend:
- Node.js
- Express
- MongoDB
- Mongoose

---

## Features

- Teacher selection
- Weekly performance chart
- Data table view
- Clean dashboard layout
- REST API integration

---

## How to Run

Clone the repository:

git clone https://github.com/Gupta5-5-2005672/savra-dashboard.git

Frontend:

cd frontend  
npm install  
npm run dev  

Backend:

cd Backend  
npm install  
node server.js
